# Online Air Ticket Booking System

* An online airline ticket system using Java Spring MVC and Hibernate
* Build tool used - Apache Maven
* Database vendor used - MySQL (Skipped for Jenkins pipeline testing)
* All JavaMail functionality upcoming

### Maven Configuration

**Add to Jenkins UI:**
1. Navigate to **Manage Jenkins** > **Tools**.
2. Under **JDK installations**, click **Add JDK**.
   * **Name:** `JDK 8`
   * **JAVA_HOME:** `/usr/lib/jvm/java-1.8.0-openjdk`
   * *Uncheck "Install automatically"*
3. Under **Maven installations**, click **Add Maven**.
   * **Name:** `Maven 3`
   * *Set path to `/usr/share/maven` or select "Install automatically"*
